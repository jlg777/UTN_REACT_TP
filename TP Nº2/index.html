Ejercicio 1

Tengo desarrollada la siguiente función en javascript:
function suma (a,b,callback){
let c = a +b;
callback();
}

¿Que es el tercer parámetro recibido?
El tercer parametro "callback" es una funcion que se llama cuando se completa la ejecución de la función principal. En este caso, el callback se llama después de que se realiza la suma de los dos números.

¿En que casos es obligatorio desarrollar este tipo de funciones?
Este tipo de funciones son obligatorias en los casos en los que se necesita realizar una acción después de que la función principal haya terminado de ejecutarse. Por ejemplo, si queremos mostrar un mensaje en la pantalla después de realizar una suma, podemos utilizar un callback.
